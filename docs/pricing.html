料金プラン

トップ / 料金プラン

従業員数と業務内容にあわせた、わかりやすい月額顧問料。
必要な業務だけを選んでご契約いただけます。

※ 下記は月額顧問料（税込）の目安です。実際の料金は業務範囲・状況に応じてお見積りいたします。

01

顧問料（月額）

ご契約は、以下の4つの業務から必要なものをお選びいただけます。従業員数に応じて、月額顧問料が決まります。

Plan A

相談

日常の労務相談・法改正情報のご案内を中心とした、もっともシンプルなプラン。

Plan B

相談・手続き業務

労務相談に加え、入退社・算定基礎・労保更新などの手続き代行を含むプラン。

Plan C

給与計算

毎月の給与計算・明細作成・振込データ作成などをお引き受けするプラン。

Plan D

給与・勤怠計算

給与計算に加え、勤怠データの集計・管理までを一括でお任せいただけるプラン。

02

スポット業務

顧問契約をされていないお客様も、単発（スポット）でのご依頼を承っております。以下は一例です。料金は業務量や内容により異なりますので、個別にお見積りいたします。

就業規則の新規作成・改定 算定基礎届・労保年度更新 入退社手続き 労災手続き 傷病手当金・育児介護休業給付手続き 助成金の申請サポート 各種労務相談（単発） 問題社員・退職・ハラスメント対応 労働基準監督署の調査対応

料金についての注意事項

  • 表示価格はすべて消費税込みの目安料金です。
  • 顧問料は、従業員数・業務範囲・業種の複雑度により変動する場合がございます。詳しくはお見積りをご依頼ください。
  • 手続き業務・給与計算に伴う実費（電子申請・郵送費等）は別途となる場合があります。
  • スポット業務（単発のご依頼）の料金は、内容を伺ったうえで個別にお見積りいたします。
  • 助成金の成果報酬部分は、受給確定後の後払いとなります。
